# Fut-Time CR — Demo web

Landing de una sola página para **Fut-Time Tournament**, organización de torneos de fútbol amateur en Heredia, Costa Rica.

> Demo construido por **Nuva IA** — el footer lleva "Powered by Nuva IA".

## Estilo
Tema "Partido Nocturno": foto de estadio de fondo, tarjetas glass oscuras y acento **lime** neón. Tipografías Anton + Manrope.

## Archivos
- `index.html` — la landing completa (hero, modalidades, cómo funciona, countdown del torneo, beneficios, momentos, testimonios, patrocinadores, inscripción, footer).
- `app.css` — estilos.
- `app.js` — countdown, menú móvil, animaciones de scroll y formulario.
- `assets/` — logo (PNG), foto de fondo del estadio y logo de Nuva.

## Cómo verlo
Abrí `index.html` con doble clic (no necesita build) o servilo con `python3 -m http.server`.

## Datos a personalizar (placeholders del demo)
- WhatsApp: `50688888888` → número real (en todos los `wa.me/...`).
- Correo: `info@futtime.cr`.
- Fecha del torneo: en `app.js` → `initCountdown()` (`2026-07-26`).
- Precios, stats, testimonios, patrocinadores y "momentos" son de ejemplo, editables en `index.html`.
