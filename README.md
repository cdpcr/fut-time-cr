# Fut-Time CR — Demo web

Sitio demo para **Fut-Time Tournament**, organización de torneos de fútbol en Heredia, Costa Rica.

> Demo construido por **Nuva IA** — el footer lleva "Powered by Nuva IA".

## Páginas
- `index.html` — Inicio: hero, cuenta regresiva al próximo torneo, beneficios, patrocinadores.
- `formato.html` — Modalidades (Fut5/7/11), categorías, precios, premios y reglas.
- `calendario.html` — Fixture por jornada, tabla de posiciones y goleadores (datos de ejemplo).
- `inscripcion.html` — Formulario de pre-inscripción + contacto (WhatsApp, IG, FB).

## Stack
HTML + CSS + JavaScript puro. **Sin build**: abrí `index.html` con doble clic.

## Datos a personalizar (placeholders del demo)
